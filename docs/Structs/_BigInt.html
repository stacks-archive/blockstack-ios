_BigInt

public struct _BigInt<Word: FixedWidthInteger & UnsignedInteger> :
    BinaryInteger, SignedInteger, CustomStringConvertible,
    CustomDebugStringConvertible
    where Word.Magnitude == Word

A dynamically-sized signed integer.

The _BigInt type is fully generic on the size of its word – the BigInt alias uses the system’s word-sized UInt as its word type, but any word size should work properly.
